# github-cpanel-ftp-deploy-guide
# GitHub Actions ile cPanel FTP Deploy Kurulumu

Bu rehber, SSH veya terminal erişimi olmayan cPanel hostinglerde GitHub Actions kullanarak otomatik FTP Deploy kurulumu yapmak isteyenler için hazırlanmıştır.

## Desteklenen Yapılar

- PHP
- Laravel
- Flutter Web
- HTML/CSS/JS
- cPanel Shared Hosting

## Özellikler

- Otomatik deploy
- .env koruması
- uploads klasörü koruması
- APK ve ZIP dosyalarını hariç tutma
- GitHub Actions entegrasyonu
